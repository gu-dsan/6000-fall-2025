Schedule

Here’s your roadmap for the semester!

  • Content (): This page contains the lecture slides for the week.

  • Lab (): This page contains lab for the week. This is usually a step-by-step instruction accompanied by a GitHub repo. A GitHub classroom link would be provided, once you accept an assignment (in GitHub classroom parlance everything is an assignment) a new repo would be created for you to work in. You will be starting this activity in class and submitting it before the due-date (Wednesday of next week 11:59PM unless otherwise stated).

  • Assignment (): This page contains the instructions for each assignment. Similar to the labs, there would be a GitHub classroom link provided to you for each assignment. Unlike the labs the assignments are more detailed and require significantly more coding effort. Assignments are also due by Wednesday of next week 11:59PM unless otherwise stated.
